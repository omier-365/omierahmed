# ⚡ مرجع سريع

## 🎯 أهم 3 أشياء

### 1. رقم الهاتف
```
ابحث في index.html عن: +20 102 263 9193
غيّره 3 مرات
```

### 2. روابط السوشيال
```html
<a href="#" class="social-icon">
غير # لرابطك الحقيقي
```

### 3. الصورة الشخصية
```html
<img src="https://via.placeholder.com/400x400...">
غيرها لـ: src="images/profile.jpg"
```

---

## 🚀 رفع على Cloudflare

1. https://dash.cloudflare.com/
2. Workers & Pages > Create
3. Upload: `index.html`, `style.css`, `main.js`
4. Deploy!

---

## 🎨 تغيير اللون الرئيسي

في `style.css` أول سطر:
```css
--color-emerald: #00FF88;  /* غير هذا */
```
