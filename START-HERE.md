## ⚡ البداية السريعة (5 دقائق)

### الخطوة 1: اختبر الموقع محلياً

1. افتح `index.html` مباشرة في المتصفح
2. شوف الموقع شغال صح؟ ✅
3. جرب على الموبايل (F12 > Toggle Device)

### الخطوة 2: غيّر المعلومات الأساسية

افتح `index.html` وغير:

```html
<!-- ابحث عن هذه الأسطر وعدلها -->

<!-- رقم الهاتف -->
+20 102 263 9193  →  رقمك

<!-- روابط واتساب -->
https://wa.me/201022639193  →  https://wa.me/رقمك

<!-- روابط السوشيال ميديا -->
<a href="#">  →  <a href="رابطك">
```

### الخطوة 3: انشر الموقع

**أسهل طريقة:**
1. اذهب: https://dash.cloudflare.com/
2. Workers & Pages > Create > Upload assets
3. ارفع: `index.html`, `style.css`, `main.js`
4. Deploy!

**رابط موقعك:** `your-project.pages.dev` 🎉

## 🎯 أول 3 تعديلات ضرورية

### ✅ 1. غيّر رقم الهاتف

في `index.html`، ابحث عن `+20 102 263 9193` (3 مواضع)

### ✅ 2. غيّر روابط السوشيال ميديا

في `index.html`، ابحث عن:
```html
<a href="#" class="social-icon">
```
غير `#` لروابطك الحقيقية

### ✅ 3. غيّر الصورة الشخصية

في `index.html`:
```html
<img src="https://via.placeholder.com/400x400..." alt="Omier Ahmed">
```

غيرها لـ:
```html
<img src="images/profile.jpg" alt="Your Name">
```

وضع صورتك في مجلد `images/`
