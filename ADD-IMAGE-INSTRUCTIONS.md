# 📸 كيف تضيف صورة "هاكر التاريخ"

## ✅ تم تجهيز الكود!

الكود جاهز في الموقع، كل اللي محتاجه:

---

## 📋 الخطوة الوحيدة:

### ضع صورة "هاكر التاريخ" في المكان الصحيح:

1. **احفظ الصورة** اللي أرسلتها باسم:
   ```
   teacher-design.jpg
   ```

2. **ضعها في المجلد:**
   ```
   d:\مشروع\images\portfolio\teacher-design.jpg
   ```

3. **خلاص!** ✅

---

## 🎯 البنية الصحيحة:

```
d:\مشروع\
├── index.html
├── style.css
├── script.js

    └── portfolio/
        └── teacher-design.jpg  ← الصورة هنا
```

## 🧪 اختبر:

1. افتح `index.html`
2. اذهب لقسم "معرض الأعمال"
3. اضغط على فلتر "مشاريع تعليمية"
4. **ستظهر الصورة!** 🎉

---

## 📝 معلومات المشروع:

- **العنوان**: تصميم تعليمي - هاكر التاريخ
- **الفئة**: تصميم للمدرسين
- **الوصف**: تصميم احترافي للأستاذ علاء الدين العراقي - برنامج "هاكر التاريخ" يجمع بين التاريخ والتكنولوجيا بطريقة إبداعية

## ❓ إذا الصورة ما ظهرت:

1. ✅ اسم الملف صحيح: `teacher-design.jpg` (حروف صغيرة)
2. ✅ المجلد موجود: `images/portfolio/`
3. ✅ الصورة في المكان الصحيح
4. ✅ حدّث المتصفح (F5 أو Ctrl+F5)
