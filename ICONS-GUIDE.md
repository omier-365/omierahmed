# 🎨 دليل الأيقونات - Font Awesome

الموقع يستخدم **Font Awesome 6.5.1** - مكتبة أيقونات ضخمة

## 🔍 كيف تغير الأيقونات؟

ابحث في الكود عن:
```html
<i class="fas fa-ICON-NAME"></i>
```

غير `fa-ICON-NAME` لاسم الأيقونة الجديدة

## 📖 ابحث عن المزيد

### مواقع للبحث عن أيقونات:

1. **Font Awesome**
   - https://fontawesome.com/icons
   - ابحث عن أي أيقونة
   - انسخ الكود

2. **أمثلة البحث:**
   - "camera" → أيقونات كاميرا
   - "shopping" → أيقونات تسوق  
   - "tech" → أيقونات تقنية

## 🎨 أنواع الأيقونات

### Solid (fas):
```html
<i class="fas fa-star"></i>
```
الأكثر استخداماً - مليئة

### Regular (far):
```html
<i class="far fa-star"></i>
```
فارغة من الداخل

### Brands (fab):
```html
<i class="fab fa-facebook"></i>
```
للشبكات الاجتماعية والشركات

---

## ✅ نصائح

1. ✅ استخدم أيقونات واضحة ومفهومة
2. ✅ حافظ على تناسق الأيقونات (كلها solid أو كلها regular)
3. ✅ لا تبالغ في استخدام الأيقونات
4. ✅ تأكد أن الأيقونة تعبر عن المحتوى
5. ✅ جرب الأيقونة على خلفية داكنة قبل الاستخدام
