# 🤖 دليل Telegram Bot الكامل

## ✅ الكود جاهز بالفعل!

الموقع يدعم Telegram Bot الآن! كل اللي محتاجه:
1. إنشاء البوت
2. وضع Token و Chat ID
3. اختبار!

---

## 📋 الخطوة 1: إنشاء البوت (5 دقائق)

### 1.1 افتح Telegram وابحث عن:
```
@BotFather
```

### 1.2 ابدأ محادثة واكتب:
```
/newbot
```

### 1.3 سيسألك عن اسم البوت:
```
اكتب: Omier Ahmed Contact Bot
(أو أي اسم تريده)
```

### 1.4 سيسألك عن Username:
```
اكتب: omierahmed_contact_bot
(يجب ينتهي بـ _bot أو bot)
```

### 1.5 ستحصل على رسالة:
```
Done! Congratulations on your new bot.

Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz123456789
```

**احفظ هذا التوكن! 👆**

---

## 📋 الخطوة 2: الحصول على Chat ID (دقيقتين)

### الطريقة السهلة:

1. **ابحث في Telegram عن:**
   ```
   @userinfobot
   ```

2. **ابدأ محادثة** معاه بـ `/start`

3. **هيرسل لك:**
   ```
   Your user ID: 123456789
   ```

4. **هذا هو Chat ID بتاعك!** احفظه

### الطريقة البديلة:

1. أرسل أي رسالة لبوتك الجديد (مثلاً: مرحباً)
2. افتح في المتصفح:
   ```
   https://api.telegram.org/bot<TOKEN>/getUpdates
   ```
   غير `<TOKEN>` بالتوكن بتاعك

3. ابحث في النتيجة عن:
   ```json
   "chat":{"id":123456789
   ```

4. الرقم `123456789` هو Chat ID

---

## 📋 الخطوة 3: إضافة البيانات للموقع (دقيقة)

### 3.1 افتح ملف `main.js`

### 3.2 ابحث عن أول سطرين (السطر 6 و 7):
```javascript
const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN_HERE';
const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID_HERE';
```

### 3.3 غيرهم لبياناتك:
```javascript
const TELEGRAM_BOT_TOKEN = '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz123456789';
const TELEGRAM_CHAT_ID = '123456789';
```

### 3.4 احفظ الملف!

### 4.5 افتح Telegram:

👤 الاسم: اختبار

## 🔧 استكشاف الأخطاء

### ❌ المشكلة: Console يقول "Telegram Bot not configured"

**السبب:** ما حطيت Token و Chat ID

**الحل:**
1. راجع `main.js`
2. تأكد أن البيانات صحيحة
3. احذف `'` من حول YOUR_BOT_TOKEN_HERE

---

### ❌ المشكلة: "Failed to send message"

**السبب:** Token أو Chat ID غلط

**الحل:**
1. راجع Token (لازم يكون فيه `:` في النص)
2. راجع Chat ID (لازم يكون أرقام فقط)
3. تأكد أنك أرسلت `/start` للبوت قبل

---

### ❌ المشكلة: CORS Error في Console

**السبب:** المتصفح بيمنع الـ request للأمان

**الحل:**
لا تقلق! لما تنشر الموقع على Cloudflare Pages، المشكلة دي هتختفي تلقائياً ✅

**للاختبار المحلي:**
- استخدم Live Server في VS Code
- أو انشر الموقع واختبره مباشرة

---

### ❌ المشكلة: البوت لا يستقبل شيء

**الحل:**
1. تأكد أنك فتحت البوت وأرسلت `/start`
2. جرب تبعت رسالة يدوياً للبوت
3. تأكد أن Chat ID صحيح

## 🎨 تخصيص الرسالة

### إذا أردت تغيير شكل الرسالة:

1. **افتح `main.js`**
2. **ابحث عن:**
   ```javascript
   const message = `
   📩 طلب مشروع جديد
   ...
   ```

3. **عدّل كما تريد!**

### أمثلة للتخصيص:

**إضافة إيموجي:**
```javascript
const message = `
🎨 طلب تصميم جديد! 🚀

👤 العميل: ${data.fullName}
...
```

**إضافة رابط واتساب:**
```javascript
const message = `
📩 طلب مشروع جديد

👤 الاسم: ${data.fullName}
📱 الهاتف: ${data.phone}
📞 واتساب: https://wa.me/${data.phone.replace(/\D/g, '')}
...
```

**إضافة رابط البريد:**
```javascript
const message = `
📩 طلب مشروع جديد

👤 الاسم: ${data.fullName}
📧 البريد: ${data.email}
✉️ رد سريع: mailto:${data.email}
...
```
